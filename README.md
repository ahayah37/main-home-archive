# Main Home Archive

## 项目简介

`main-home-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。这里不针对任何特定域名或网站，仅作为静态页面的统一存放与版本管理空间。所有页面均为独立文件，彼此之间无强关联，便于索引、引用或离线查看。

## 目录结构

```
main-home-archive/
├── pages/          # 独立 HTML 页面文件（按时间或主题组织）
├── assets/         # 页面依赖的公共资源（CSS、JS、图片等）
├── index.html      # 仓库首页索引，列出所有归档页面
└── README.md       # 本文件
```

- `pages/`：存放各个独立的 HTML 文件，文件名通常反映内容主题或归档日期。
- `assets/`：若多个页面共享样式或脚本，可放置于此，减少重复。
- `index.html`：一个简单的导航页，列出所有归档页面及其简要说明。

## 页面归档说明

- 每个页面均为独立完整的 HTML 文件，可直接在浏览器中打开，无需额外服务。
- 归档页面可能包含实验性内容、个人项目展示、学习笔记或临时记录等，不保证长期更新或维护。
- 页面内不包含任何外部资源链接（除本仓库 `assets/` 目录外），以确保离线可用性。

## 维护说明

- 本仓库由维护者不定期更新，新增页面将直接放入 `pages/` 目录并更新 `index.html`。
- 欢迎通过 Issue 提交建议或指出页面问题，但请注意本仓库不针对具体网站，不接受非归档类请求。
- 不推荐直接 Fork 后用于商业或营销目的，本仓库仅作为个人或协作归档用途。
